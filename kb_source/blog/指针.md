# 数组

## 数组名

源码
```c
#include <stdio.h>

int main()
{
    int a[]={1,2,3,5,6};
    int *p1=&a[0];
    int *p2=&a;
    int *p3=&a[1];
    int *p4=a;
    printf("p1=&a[0],%p\n",p1);
    printf("p2=&a,%p\n",p2);
    printf("p3=&a[1],%p\n",p3);
    printf("p4=a,%p\n",p4);
}
```
输出
```
p1=&a[0],0x7fffd4c3fb90
p2=&a,0x7fffd4c3fb90
p3=&a[1],0x7fffd4c3fb94
p4=a,0x7fffd4c3fb90
```
但是在编译时会出现警告：
```
new.c: In function ‘main’:
new.c:15:13: warning: initialization of ‘int *’ from incompatible pointer type ‘int (*)[5]’ [-Wincompatible-pointer-types]
   15 |     int *p2=&a;
      |             ^
```
错误是类型不匹配。查了一下，是因为a是一个指针，类型是`int *`，而`&a`是一个数组的地址，类型是`int (*)[5]`，不能直接赋值给`int *`指针，即使两者的内容是一样的。

由上可见：`&a[0]`，`a`都是指向数组首地址的指针，`&a`也是指向数组首地址的指针，但是类型是`int (*)[5]`；`a[1]`与*(a+1)是一样的。

## 指针数组

源码
```c
#include <stdio.h>

int main()
{
    char *a[]={"one","two","three"};
    char *p1=a;
}
```
编译后会出现警告：
```
new2.c:6:14: warning: initialization of ‘char *’ from incompatible pointer type ‘char **’ [-Wincompatible-pointer-types]
    6 |     char *p1=a;
      |              ^
```
错误·是类型不匹配。原因是`a`是一个指针，指向一个指针数组，这个指针数组里的内容也是指针，所以`a`是一个指向指针的指针，即二级指针，所以类型是`char **`，不能直接赋值给`char *`指针p1，将`char *p1=a`改为`char **p1=a`即可。

打印这个指针数组指向的字符串
```c
#include <stdio.h>

int main()  
{  
    char *a[]={"one","two","three"};  
    char **p1=a;  
    int i;  
    for(i=0;i<3;i++)  
    {  
        printf("%s\n",a[i]);  
        printf("%s\n",*p1[i]);  
        printf("%s\n",*(p1+i));
    }  
}  
```

编译后出现警告：
```
new2.c:11:18: warning: format ‘%s’ expects argument of type ‘char *’, but argument 2 has type ‘int’ [-Wformat=]
   11 |         printf("%s\n",*p1[i]);
      |                 ~^    ~~~~~~~~
      |                  |    |
      |                  |    int
      |                  char *
      |                 %d
```
`*p1[i]`相当于是`*(*(p1+i))`，`(p1+0)`是一个指向字符串首地址的指针，那么`*(p1+0)`里的内容已经是字符串的首地址了，再解引用那么内容就是字符串里的首个字符，所以`*p1[i]`的内容就是字符串里的首个字符。

要打印字符串，用的是`%s`,后面的参数应该是这个字符串的首地址，所以出现警告。

可以改下程序验证一下：

```c
#include <stdio.h>

int main()  
{  
    char *a[]={"one","two","three"};  
    char **p1=a;  
    int i;  
    for(i=0;i<3;i++)  
    {  
        printf("a[i] %s\n",a[i]);  
        printf("*p1[i] %c\n",*p1[i]);  
        printf("*(p1+i) %s\n",*(p1+i));
    }  
}  
```

那么打印的应该是这三个字符串的首个字符`o`，`t`，`t`。

输出
```
a[i] one
*p1[i] o
*(p1+i) one
a[i] two
*p1[i] t
*(p1+i) two
a[i] three
*p1[i] t
*(p1+i) three
```
确实如此。

# 指针的大小

指针存放的内容是地址，不同的系统的地址范围不同，与位数有关。32位系统下，指针大小为4字节，即32位，64位系统下，指针大小为8字节。
